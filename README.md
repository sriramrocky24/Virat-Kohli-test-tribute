# Virat-Kohli-test-tribute
A fan-made tribute to Virat Kohli’s legendary Test cricket career using SQL analysis, Python scripting, and Power BI visualizations.

# Overview
This dashboard shows Virat Kohli's Test cricket performance across all opponents

Data includes:
- Career stats (9,230 runs, 46.85 average, 55.58 strike rate)
- Performance vs 7 opponents (Australia, South Africa, New Zealand, England, Sri Lanka, West Indies and Bangladesh)
- Milestones (30 centuries, 7 double centuries, 31 fifties)
- Boundaries (1,027 fours, 30 sixes)

# Dashboard Structure

**Tab 1: Career Insights**
- KPI cards (matches, innings, runs, average, strike rate)
- Chart 1: Runs vs Balls Faced (shows efficiency by opponent)
- Chart 2: Average vs Strike Rate (shows playing style by opponent)
- Filter: Select opponent to explore

**Tab 2: Legendary Strokeplay**
- KPI cards (fifties, hundreds, double hundreds, fours, sixes)
- Chart 3: Fifties/Hundreds distribution (team-wise)
- Chart 4: Fours/Sixes distribution (team-wise)
- Filter: Select opponent

# Key Findings

**1. Best Performance: Sri Lanka**
- Average: 67.81 (highest)
- Strike Rate: 68.28% (highest)
- Insight: Weak bowling allows aggressive + consistent batting

**2. Toughest Opponent: Australia**
- Average: 43.76 (lowest)
- Strike Rate: 52.00% (controlled)
- Balls Faced: 4,292 (most innings)
- Insight: Elite bowling forces defensive approach

**3. Playing Style**
- Fours: 1,027 (97.1% of boundaries)
- Sixes: 30 (2.9% of boundaries)
- Ratio: 34:1 (technical, not aggressive)
- Insight: Orthodox Test batsman, emphasizes ground shots

**4. Milestone Conversion**
- Fifties: 31
- Centuries: 30
- Conversion: 97% (near-perfect)
- Insight: Rarely gets out between 50-99 runs

**5. Consistency**
- All 7 opponents covered (no weak matchups)
- Career average 46.85 = elite level
- Shows: Adaptable across different bowling styles
